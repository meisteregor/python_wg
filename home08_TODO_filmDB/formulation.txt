## Домашнее задание

Консольный клиент к базе денных фильмов.

В качестве примера интерфейся можно рассмотреть sql - https://sqlite.org/cli.html

Информация о фильмах хранится в строковой переменной TABLE_CSV.

Поддерживаются следующие специальные команды:

.help показать справку
.exit выход из программы

Поддерживается следующий язык запросов.

Отобразить все столбцы всех данные.

SELECT * FROM TABLE;

Отобразить все столбцы всех данных отсортированные по столбцу column в порядке возрастания.
SELECT * FROM TABLE ORDER BY column ASC;

Отобразить все столбцы всех данных отсортированные по столбцу column в порядке убывания.
SELECT * FROM TABLE ORDER BY column DESC;

ЗАДАНИЕ:

Реализовать нереализованные функции, так чтобы программа заработвала в соответствии документацией.
