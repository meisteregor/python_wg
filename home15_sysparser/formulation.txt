Напишите скрипт для обработки системных логов.
Логи представляют собой текстовый файл, каждая строка содержит одно сообщение следующего формата:
timestamp: level: program: message
timestamp - дата и время сообщения
level - уровень сообщения, например, ERROR, WARN, DEBUG
program - программа, написавшая сообщение
message - текст сообщения
Программа имеет следующий интерфейс:
./logproc.py --help
usage: logproc.py [-h] [--level LEVEL] [--app APP] [--output OUTPUT] input
positional arguments:
  input            входоной файл логов
optional arguments:
  -h, --help       show this help message and exit
  --level LEVEL    уровень сообщения, который надо отфильтровать, если не
                   задано - все уровни
  --app APP        название программы, которую надо отфильтровать, если не
                   задано - все программы
  --output OUTPUT  выходной файл, если не задано на стандартный вывод
Ошибки обработать.
Примеры граничных условий:
    Невалидные входные параметры - файл не существует, передали директорию вместо файла
    Невалидный файл логов - формат не соответствует докуменатции.
В этом случае прекращать работу программы.
Рекомндации:
    Самое простое решение - самое лучшее;
    Делайте поэтапно:
        Начните с файла data01.txt
        чтение файла, вывод его на экран
        добавьте фильтарцию по уровню
        добавьте фильтарцию по имени програмы
        добавьте вывод в файл
        добавьте проверьте граничные условия - файлы data02.txt, data03.txt
